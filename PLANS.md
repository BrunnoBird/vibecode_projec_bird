# PLANS

## Visão
Criar um projeto Android escalável, modular e preparado para evolução orientada por especificação.

## Fase 1 — Bootstrap técnico
- Estruturar Gradle
- Criar módulos base
- Configurar Compose
- Configurar Material 3
- Definir baseline MVI
- Criar navegação inicial
- Habilitar testes unitários

## Fase 2 — Fundamentos de produto
- Definir primeira feature real
- Escrever PRD
- Derivar spec
- Quebrar em tasks
- Implementar via agentes

## Fase 3 — Escala
- Adicionar convenções de logging
- Adicionar tratamento padrão de erro
- Adicionar DI
- Adicionar CI
- Adicionar testes instrumentados quando necessário
