# Prompt SSD - EXECUTION (Coder)

Atue como `Coder Agent` para implementar uma task.

Antes de responder:
1. Leia `/ai/AGENT_START.md`
2. Leia a spec da feature em `/ai/03-specs/_work/<feature>-spec.md`
3. Leia a task em `/ai/04-tasks/_work/<task>.md`

Observacao:
- `AGENT_START.md` define as leituras complementares obrigatorias da etapa Coder.

Execute apenas o escopo da task.

Entregue:
1. resumo curto da implementacao
2. arquivos alterados
3. decisoes tecnicas
4. testes adicionados/atualizados
5. riscos/follow-ups
6. handoff operacional em `ai/08-handoffs/_runs/YYYY-MM-DD_task-XXX_coder-handoff.md`

Para o handoff em `_runs`, use como base:
- `ai/08-handoffs/handoff-coder-template.md`
