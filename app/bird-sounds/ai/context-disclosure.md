# Context Disclosure — bird-sounds

> Sub-projeto de `app/bird-sounds`. Leia também `ai/02-context/context-disclosure.md` (base global).

## Domínio
Focado no reconhecimento, reprodução e gravação de cantos de pássaros, servindo como uma biblioteca de áudio e possível ferramenta de machine learning de áudio para classificação.

## Features Específicas
- Player de áudio especializado para espectrogramas de sons de aves
- Captura de áudio no microfone para registro de cantos 
- Integração potencial com engine de reconhecimento acústico

## Dependências Únicas
- ExoPlayer / Media3 (para reprodução de áudio robusta)
- Bibliotecas para processamento e gravação de áudio nativo (placeholder)

## UI (sobrescreve global)
- Jetpack Compose focado em visualização de espectrogramas e controles de mídia dinâmicos.
